## 🎯 Do These 4 Things Right Now:

### ✅ Step 1: Enable GitHub Actions (30 seconds)

**Click this link:** https://github.com/riginoommen/riginoommen/settings/actions

**Do these 3 things:**
1. Under "Actions permissions" → Select **"Allow all actions and reusable workflows"**
2. Under "Workflow permissions" → Select **"Read and write permissions"**
3. Check ✅ **"Allow GitHub Actions to create and approve pull requests"**
4. Click **Save**

---

### ✅ Step 2: Enable Auto-Merge (15 seconds)

**Click this link:** https://github.com/riginoommen/riginoommen/settings

**Scroll to "Pull Requests" section:**
1. Check ✅ **"Allow auto-merge"**
2. Click **Save**

---

### ✅ Step 3: Run All Workflows (2 minutes)

**Click this link:** https://github.com/riginoommen/riginoommen/actions

**Run each workflow ONCE** (they'll auto-run after this):

**3.1 Latest Blog Posts**
- Click "Latest Blog Posts" in left sidebar
- Click "Run workflow" button (top right)
- Click "Run workflow" in dropdown
- ✅ Wait ~30 seconds, it should turn green

**3.2 GraphQL Foundation Blog Feed**
- Click "GraphQL Foundation Blog Feed" in left sidebar
- Click "Run workflow" button
- Click "Run workflow"

**3.3 Update GitHub Activity**
- Click "Update GitHub Activity" in left sidebar
- Click "Run workflow" button
- Click "Run workflow"
- ✅ Wait ~30 seconds

**3.4 Check Links**
- Click "Check Links" in left sidebar
- Click "Run workflow" button
- Click "Run workflow"
- ✅ Wait ~30 seconds

## 🔍 Quick Troubleshooting

### "Blog posts not showing?"

**Option 1:** You might not have published Medium articles yet
- Check: https://medium.com/@riginoommen
- Publish an article and re-run workflow

**Option 2:** Verify feed works
```bash
bash .github/verify-feeds.sh
```

### "Workflow failed?"

1. Click on the failed workflow in Actions tab
2. Read the error message
3. Common fix: Re-run the workflow (button in top right)

### "Activity not showing?"

- Make sure you set "Read and write permissions"
- Re-run "Update GitHub Activity" workflow
